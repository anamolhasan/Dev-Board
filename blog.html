Question-1: What are the different ways to select an element in the DOM?

JavaScript ব্যবহার করে DOM-এর একটি এলিমেন্ট নির্বাচন করার জন্য বিভিন্ন পদ্ধতি রয়েছে। নিচে কিছু গুরুত্বপূর্ণ পদ্ধতি দেওয়া হলো:

  1. getElementById – ID এর মাধ্যমে নির্দিষ্ট একটি এলিমেন্ট নির্বাচন করা হয়।
  2. getElementsByClassName – একটি নির্দিষ্ট ক্লাসের সমস্ত এলিমেন্ট নির্বাচন করা হয়।
  3. getElementsByTagName – নির্দিষ্ট ট্যাগ নামের সমস্ত এলিমেন্ট নির্বাচন করা হয়।
  4. querySelector – CSS সিলেক্টরের মাধ্যমে প্রথম ম্যাচিং এলিমেন্ট নির্বাচন করা হয়।
  5. querySelectorAll – CSS সিলেক্টরের মাধ্যমে সমস্ত ম্যাচিং এলিমেন্ট নির্বাচন করা হয়।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, এবং textContent এর মধ্যে পার্থক্য হলো:

  • innerHTML: এটি একটি এলিমেন্টের সম্পূর্ণ HTML কন্টেন্ট রিটার্ন করে, যার মধ্যে HTML ট্যাগও অন্তর্ভুক্ত থাকে।
  • innerText: এটি শুধুমাত্র এলিমেন্টের দৃশ্যমান (visible) টেক্সট রিটার্ন করে এবং যেসব টেক্সট CSS দ্বারা লুকানো (display: none;) থাকে সেগুলো বাদ দেয়।
  • textContent: এটি এলিমেন্টের সমস্ত টেক্সট রিটার্ন করে, যার মধ্যে হিডেন (লুকানো) টেক্সট, হোয়াইটস্পেস এবং লাইন ব্রেকও অন্তর্ভুক্ত থাকে, তবে এটি HTML ব্যাখ্যা (interpret) করে না এবং কাঁচা টেক্সট প্রদান করে।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন (Event Delegation) হল একটি JavaScript টেকনিক, যেখানে একটি প্যারেন্ট এলিমেন্টের সাথে একটি একক ইভেন্ট লিসেনার সংযুক্ত করা হয়, যা তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট পরিচালনা করতে পারে। এটি প্রতিটি চাইল্ড এলিমেন্টের জন্য আলাদা ইভেন্ট লিসেনার যোগ করার পরিবর্তে ব্যবহার করা হয়।

Question-4: What is event bubbling in the DOM?

ইভেন্ট বাবলিং (Event Bubbling) হল একটি ইভেন্ট প্রোপাগেশন মেকানিজম, যেখানে কোনো চাইল্ড এলিমেন্টে ইভেন্ট ট্রিগার হলে, সেই ইভেন্ট ধাপে ধাপে তার প্যারেন্ট এলিমেন্ট, তারপর তার প্যারেন্টের প্যারেন্ট, এভাবে document অবধি উপরের দিকে ছড়িয়ে পড়ে।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript দিয়ে DOM-এ এলিমেন্ট তৈরি, যোগ, এবং মুছে ফেলা একটি সাধারণ কাজ।

নতুন এলিমেন্ট তৈরি:

  • let newElement = document.createElement("tagName");

    • DOM-এ যোগ করা:

  • parent.appendChild(newElement); (শেষে যোগ)
  • parent.insertBefore(newElement, referenceElement); (নির্দিষ্ট স্থানে যোগ)

    • DOM থেকে এলিমেন্ট মুছে ফেলা:

  • parent.removeChild(child); (প্যারেন্ট থেকে চাইল্ড মুছতে)
  • element.remove(); (সরাসরি এলিমেন্ট মুছতে)
